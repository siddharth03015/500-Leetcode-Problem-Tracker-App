# 500 DSA Problem Tracker 🚀

A sleek, modern, and interactive web application designed to help developers track their progress as they solve the top 500 curated Data Structures and Algorithms (DSA) problems on LeetCode. 

Built with **Next.js**, **React**, **Tailwind CSS**, and **Framer Motion**, this application features a premium dark-mode aesthetic, smooth scrolling, and dynamic micro-animations.

## Features ✨

- **Curated Problem List**: Access a hand-picked list of 500 high-yield LeetCode problems categorized by difficulty (Easy, Medium, Hard).
- **Local Progress Tracking**: Mark problems as solved, and your progress is securely saved in your browser's local storage.
- **Dynamic Progress Dashboard**: View your overall completion percentage and a breakdown of problems solved by difficulty level.
- **Community Wall of Fame**: A dedicated review section. Once you solve all 500 problems, the review form unlocks, allowing you to leave your mark on the Wall of Fame!
- **Share Your Milestone**: One-click social sharing buttons (LinkedIn, Twitter) that dynamically generate posts with your current solved count.
- **Premium UI/UX**: Custom cursors, smooth scrolling (Lenis), noise overlays, and beautiful layout components powered by Framer Motion.

## Tech Stack 🛠️

- [Next.js](https://nextjs.org/) (App Router)
- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Lenis](https://lenis.darkroom.engineering/) (Smooth Scrolling)

## Getting Started 💻

To run this project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/siddharth03015/500-Leetcode-Problem-Tracker-App.git
   cd 500-Leetcode-Problem-Tracker-App
   ```

2. **Install dependencies:**
   ```bash
   npm install --legacy-peer-deps
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the app in action!
